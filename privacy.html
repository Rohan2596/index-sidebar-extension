1. Overview

Finzo – Real-time Index Watchlist does not collect, store, or share any personal user data.

2. Data Collection

This extension does not collect any personally identifiable information (PII). No user data is tracked, stored, or transmitted.

3. External Data Sources

Market data is fetched from a publicly available Google Sheets document using a CSV endpoint. This data is read-only and used solely to display market information inside the extension.

4. Permissions Usage

The extension uses the following permissions:

  • sidePanel: Display the main UI panel
  • storage: Reserved for future settings
  • host_permissions: Access Google Sheets data source

5. Data Sharing

We do not sell, trade, or share any user data with third parties.

6. Security

Since no personal data is collected, risks related to data breaches are minimal.
